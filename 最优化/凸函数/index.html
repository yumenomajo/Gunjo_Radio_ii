# 3. 凸函数

# 3.1.1 定义

函数 f:Rn→Rf: R^n \to Rf:Rn→R 是凸的。

# 条件：

  1. 如果 domf\text{dom} \ fdom f 是凸集，
  2. 对于任意 x,y∈domfx, y \in \text{dom} \ fx,y∈dom f 且 0≤θ≤10 \leq \theta \leq 10≤θ≤1，有：

    f(θx+(1−θ)y)≤θf(x)+(1−θ)f(y)(3.1)f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y) \tag{3.1}f(θx+(1−θ)y)≤θf(x)+(1−θ)f(y)(3.1)

# 补充：

  1. 当 ≤\leq≤ 改为 <<< 时，fff 为严格凸函数
  2. 定义 fff 为凹的，条件：−f-f−f 为凸函数
  3. 定义 fff 为严格凹的，条件：−f-f−f 为严格凸函数
  4. 某函数是仿射函数 ⟺\iff⟺ 某函数既凸也凹
  5. fff 是凸的 ⟺\iff⟺ 任意 x∈domfx \in \text{dom} \ fx∈dom f 和任意向量 vvv，函数 g(t)=f(x+tv)g(t) = f(x + tv)g(t)=f(x+tv) 是凸的（其定义域为 {t∣x+tv∈domf}\{t | x + tv \in \text{dom} \ f\}{t∣x+tv∈dom f}）

# 3.1.2 扩展值延伸

# 条件：

fˉ(x)={f(x)x∈domf∞x∉domf\bar{f}(x) = \begin{cases} f(x) & x \in \text{dom} \ f \\\\ \infty & x \notin \text{dom} \ f \end{cases}fˉ​(x)=⎩⎪⎪⎨⎪⎪⎧​f(x)∞​x∈dom fx∈/dom f​

# 定义：fˉ:Rn→R∪{∞}\bar{f}: R^n \to R \cup \{\infty\}fˉ​:Rn→R∪{∞} 对扩展值延伸。

  1. 函数延伸由确定 fff 的定义域：domf={x∣fˉ(x)<∞}\text{dom} \ f = \{x | \bar{f}(x) < \infty\}dom f={x∣fˉ​(x)<∞}。
  2. 对于任意 xxx 和 yyy，0<θ<10 < \theta < 10<θ<1，有：

    fˉ(θx+(1−θ)y)≤θfˉ(x)+(1−θ)fˉ(y)\bar{f}(\theta x + (1-\theta)y) \leq \theta \bar{f}(x) + (1-\theta)\bar{f}(y)fˉ​(θx+(1−θ)y)≤θfˉ​(x)+(1−θ)fˉ​(y)

  3. 同理可定义凹函数定义域外部为−∞对其进行延伸。

# 3.1.3 一阶条件：

大前提：在开集 domf\text{dom} \ fdom f 可微，
有性质：函数是凸函数 ⟺domf\iff \text{dom} \ f⟺dom f 是凸集且任意 x,y∈domfx, y \in \text{dom} \ fx,y∈dom f，下式成立：

f(y)≥f(x)+∇f(x)T(y−x)(3.2)f(y) \geq f(x) + \nabla f(x)^T (y - x) \tag{3.2}f(y)≥f(x)+∇f(x)T(y−x)(3.2)

# 补充：

  1. 全局下估计：
    由 f(x)+∇f(x)T(y−x)f(x) + \nabla f(x)^T (y - x)f(x)+∇f(x)T(y−x) 得出的仿射函数 yyy 即为函数 fff 在点 xxx 附近的 Taylor 近似。不等式 (3.2) 表明，对于一个凸函数，其一阶 Taylor 近似实际上是原函数的一个全局下估计。反之，如果某个函数的一阶 Taylor 近似总是其全局下估计，那么这个函数是凸的。

    不等式 (3.2) 说明从一个凸函数的局部信息（即它在某点处的函数值及导数），我们可以得到一些全局信息（如它的全局下估计）。这也许是凸函数的最重要的信息，因此对解释凸性函数以及凸优化问题的绝非常重要的性质。

    示例：

  2. 关于严格凸，凸函数的一阶条件：类似于定义中的做法。

# 一阶条件的证明：

为了证明式 (3.2)，先考虑 n=1n=1n=1 的情况。我们证明可微函数 f:R→Rf: R \to Rf:R→R 是凸函数的充要条件是对于 domf\text{dom} \ fdom f 内的任意 xxx 和 yyy，有：

f(y)≥f(x)+f′(x)(y−x).(3.4)f(y) \geq f(x) + f'(x)(y - x). \tag{3.4}f(y)≥f(x)+f′(x)(y−x).(3.4)

首先假设 fff 是凸函数，且 x,y∈domfx, y \in \text{dom} \ fx,y∈dom f，因为 domf\text{dom} \ fdom f 是凸集（某个区间），对于任意 0<t<10 < t < 10<t<1，我们有 z=x+t(y−x)∈domfz = x + t(y - x) \in \text{dom} \ fz=x+t(y−x)∈dom f，由函数 fff 的凸性可得：

f(x+t(y−x))≤(1−t)f(x)+tf(y).f(x + t(y - x)) \leq (1 - t)f(x) + tf(y).f(x+t(y−x))≤(1−t)f(x)+tf(y).

将上述两端同除以 ttt 可得：

f(y)≥f(x)+f(x+t(y−x))−f(x)t.f(y) \geq f(x) + \frac{f(x + t(y - x)) - f(x)}{t}.f(y)≥f(x)+tf(x+t(y−x))−f(x)​.

令 t→0t \to 0t→0，可以得到不等式 (3.4)。

为了证明充分性，假设对于 domf\text{dom} \ fdom f（某个区间）内的任意 xxx 和 yyy，函数满足不等式 (3.4)。选择任意 x≠yx \neq yx=y，0≤θ≤10 \leq \theta \leq 10≤θ≤1，令 z=θx+(1−θ)yz = \theta x + (1 - \theta)yz=θx+(1−θ)y。两次应用不等式 (3.4) 可得：

f(x)≥f(z)+f′(z)(x−z),f(y)≥f(z)+f′(z)(y−z).f(x) \geq f(z) + f'(z)(x - z), \quad f(y) \geq f(z) + f'(z)(y - z).f(x)≥f(z)+f′(z)(x−z),f(y)≥f(z)+f′(z)(y−z).

将第一个不等式乘以 θ\thetaθ，第二个不等式乘以 1−θ1 - \theta1−θ，并将二者相加可得：

θf(x)+(1−θ)f(y)≥f(z),\theta f(x) + (1 - \theta)f(y) \geq f(z),θf(x)+(1−θ)f(y)≥f(z),

从而说明了函数 fff 是凸的。

现在来证明一般情况，即 f:Rn→Rf: R^n \to Rf:Rn→R。设 x,y∈Rnx, y \in R^nx,y∈Rn，考虑过这两点的直线上的函数 fff，即函数 g(t)=f(ty+(1−t)x)g(t) = f(ty + (1 - t)x)g(t)=f(ty+(1−t)x)，此函数对于 ttt 求导可得 g′(t)=∇f(ty+(1−t)x)T(y−x)g'(t) = \nabla f(ty + (1 - t)x)^T (y - x)g′(t)=∇f(ty+(1−t)x)T(y−x)。

首先假设函数 fff 是凸的，则函数 ggg 是凸的，由前面的讨论可得 g(1)≥g(0)+g′(0)g(1) \geq g(0) + g'(0)g(1)≥g(0)+g′(0)，即：

f(y)≥f(x)+∇f(x)T(y−x).f(y) \geq f(x) + \nabla f(x)^T (y - x).f(y)≥f(x)+∇f(x)T(y−x).

再假设此不等式对于任意 xxx 和 yyy 均成立，因此若 t~y+(1−t~)x∈domf\tilde{t}y + (1 - \tilde{t})x \in \text{dom} \ ft~y+(1−t~)x∈dom f 以及 t~y+(1−t~)x∈domf\tilde{t}y + (1 - \tilde{t})x \in \text{dom} \ ft~y+(1−t~)x∈dom f，我们有：

即 g(t)≥g(t~)+g′(t~)(t−t~)g(t) \geq g(\tilde{t}) + g'(\tilde{t})(t - \tilde{t})g(t)≥g(t~)+g′(t~)(t−t~)，说明了函数 ggg 是凸的。

# 3.1.6 下水平集

函数 f:Rn→Rf : \mathbb{R}^n \to \mathbb{R}f:Rn→R 的 α\alphaα - 下水平集定义为：

Cα={x∈domf∣f(x)≤α}.C_\alpha = \{x \in \text{dom} \ f \ | \ f(x) \leq \alpha\}.Cα​={x∈dom f ∣ f(x)≤α}.

对于任意 α\alphaα 值，凸函数的下水平集仍然是凸集。

# 证明：

由定义得：如果 x,y∈Cαx, y \in C_\alphax,y∈Cα​，则有 f(x)≤α,f(y)≤αf(x) \leq \alpha, f(y) \leq \alphaf(x)≤α,f(y)≤α，

因此对于任意 0≤θ≤10 \leq \theta \leq 10≤θ≤1，有 f(θx+(1−θ)y)≤αf(\theta x + (1 - \theta)y) \leq \alphaf(θx+(1−θ)y)≤α，

即 θx+(1−θ)y∈Cα\theta x + (1 - \theta)y \in C_\alphaθx+(1−θ)y∈Cα​。

反过来不一定正确：某个函数的所有下水平集都是凸集，但这个函数可能不是凸函数。

例如，函数 f(x)=−e−xf(x) = -e^{-x}f(x)=−e−x 在 R\mathbb{R}R 上不是凸函数（实际上，它是严格凹函数），但是其所有下水平集均为凸集。

如果 −f-f−f 是凸函数，则由 {x∈domf∣f(x)>α}\{x \in \text{dom} \ f \ | \ f(x) > \alpha\}{x∈dom f ∣ f(x)>α} 定义的 α\alphaα - 上水平集也是凸集。

下水平集的性质可以用来判断集合的凸性，若某个集合可以描述为一个凸函数的下水平集，或者一个凹函数的上水平集，则其是凸集。

# 3.1.7 上镜图

函数 f:Rn→Rf : \mathbb{R}^n \to \mathbb{R}f:Rn→R 的图像定义为：

{(x,f(x))∣x∈domf},\{(x, f(x)) \ | \ x \in \text{dom} \ f\},{(x,f(x)) ∣ x∈dom f},

它是 Rn+1\mathbb{R}^{n+1}Rn+1 空间的一个子集。

函数 f:Rn→Rf : \mathbb{R}^n \to \mathbb{R}f:Rn→R 的上镜图定义为：

epif={(x,t)∣x∈domf,f(x)≤t}.\text{epi} \ f = \{(x, t) \ | \ x \in \text{dom} \ f, \ f(x) \leq t\}.epi f={(x,t) ∣ x∈dom f, f(x)≤t}.

它也是 Rn+1\mathbb{R}^{n+1}Rn+1 空间的一个子集。（“Epi” 是 “之上的” 意思，所以上镜图的英文 epigraph 是 “在函数图像之上” 的意思。）

凸集和凸函数的联系可以通过上掩图来建立：

  1. 一个函数是凸函数，当且仅当其上掩图是凸集。
  2. 一个函数是凹函数，当且仅当其下掩图

hypof={(x,t)∣t≥f(x)},\text{hypo} \ f = \{(x, t) \ | \ t \geq f(x)\},hypo f={(x,t) ∣ t≥f(x)},

是凸集。

# 3.1.8 Jensen 不等式及其扩展

# 基本不等式

f(θx+(1−θ)y)≤θf(x)+(1−θ)f(y),f(\theta x + (1 - \theta)y) \leq \theta f(x) + (1 - \theta)f(y),f(θx+(1−θ)y)≤θf(x)+(1−θ)f(y),
