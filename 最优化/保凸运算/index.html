# 构造性凸性

  1. 从给定的函数fff 表达式开始。
  2. 为表达式构建解析树：
    • 叶子节点为变量或常量；
    • 非叶节点为子表达式的函数。
  3. 使用复合规则为子表达式标注凸性（凸、凹、仿射或无）。
  4. 如果根节点被标记为凸（凹），则函数fff 是凸（凹）的。
  5. 扩展：为每个表达式标记符号，并结合符号相关的单调性规则。
  • 这种方法足以证明fff 是凸（凹）的，但不必要。
  • 这种验证凸性（凹性）的方法可以轻松自动化。

# 约束凸规划（Disciplined Convex Programming, DCP）

在约束凸规划（DCP）中，用户通过构造性凸性分析来构建凸函数和凹函数，表达式具有以下特点：

  • 表达式由以下部分组成：
    • 变量，
    • 常量，
    • 来自库的原子函数。
  • 原子函数具有已知的凸性、单调性和符号属性。
  • 所有子表达式均符合一般复合规则。
  • 一个有效的 DCP 函数满足以下条件：
    • 通过构造即为凸函数；
    • 语法上是凸的（可以 “局部” 验证）。

凸性依赖于原子函数的属性，而非其具体含义：

  • 例如，可以替换f\sqrt{f}f​ 与f1/4f^{1/4}f1/4，或exp⁡\expexp 与(⋅)+(\cdot)_+(⋅)+​，因为它们的属性相同。

设函数fff 是凸且可微，其定义域为 domf=Rn\text{dom} \, f = \mathbb{R}^ndomf=Rn，使 yTx−f(x)y^T x - f(x)yTx−f(x) 取最大的x∗x^*x∗ 满足 y=∇f(x∗)y = \nabla f(x^*)y=∇f(x∗)。反之，若x∗x^*x∗ 满足 y=∇f(x∗)y = \nabla f(x^*)y=∇f(x∗)，则在 x∗x^*x∗ 处达到最大值。因此，

f∗(y)=xT∇f(x)−f(x).f^*(y) = x^T \nabla f(x) - f(x).f∗(y)=xT∇f(x)−f(x).

所以，给定任意yyy，我们可以求解梯度方程 y=∇f(x)y = \nabla f(x)y=∇f(x)，从而得到 yyy 的共轭函数 f∗(y)f^*(y)f∗(y)。

换一个角度理解。若选定 z∈Rnz \in \mathbb{R}^nz∈Rn，令 y=∇f(z)y = \nabla f(z)y=∇f(z)，则：

f∗(y)=zT∇f(z)−f(z).f^*(y) = z^T \nabla f(z) - f(z).f∗(y)=zT∇f(z)−f(z).

# 伸缩变换和复合仿射变换

  1. 伸缩变换
    若 a>0a > 0a>0 以及 b∈Rb \in \mathbb{R}b∈R，则 g(x)=af(x)+bg(x) = a f(x) + bg(x)=af(x)+b 的共轭函数为：

    g∗(y)=af∗(y/a)−b.g^*(y) = a f^*(y / a) - b.g∗(y)=af∗(y/a)−b.

  2. 复合仿射变换
    若 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n 非奇异，b∈Rnb \in \mathbb{R}^nb∈Rn，则函数 g(x)=f(Ax+b)g(x) = f(Ax + b)g(x)=f(Ax+b) 的共轭函数为：

    g∗(y)=f∗(A−Ty)−bTA−Ty,g^*(y) = f^*(A^{-T}y) - b^T A^{-T}y,g∗(y)=f∗(A−Ty)−bTA−Ty,

    其定义域为：

    domg∗=ATdomf∗.\text{dom} \, g^* = A^T \text{dom} \, f^*.domg∗=ATdomf∗.

# 独立函数的和

若函数 f(u,v)=f1(u)+f2(v)f(u, v) = f_1(u) + f_2(v)f(u,v)=f1​(u)+f2​(v)，其中 f1f_1f1​ 和 f2f_2f2​ 是凸函数，且共轭函数分别为 f1∗f_1^*f1∗​ 和 f2∗f_2^*f2∗​，则：

f∗(w,z)=f1∗(w)+f2∗(z).f^*(w, z) = f_1^*(w) + f_2^*(z).f∗(w,z)=f1∗​(w)+f2∗​(z).

换言之，独立凸函数的和的共轭函数是各个凸函数的共轭函数的和。
（“独立” 是指每个函数具有不同的变量。）

# 3.3 准凸函数（Quasiconvex Functions）

  • 函数 f:Rn→Rf: \mathbb{R}^n \to \mathbb{R}f:Rn→R 是准凸的，当且仅当 domf\text{dom} \, fdomf 是凸集，且所有的下水平集：

    Sα={x∈domf∣f(x)≤α}S_\alpha = \{x \in \text{dom} \, f \mid f(x) \leq \alpha\}Sα​={x∈domf∣f(x)≤α}

    对所有 α\alphaα 都是凸集。

# 特性

  1. 如果 fff 是准凹函数，则 −f-f−f 是准凸函数。
  2. 如果 fff 同时是准凸和准凹函数，则 fff 是准线性的（quasilinear）。

# 准凸函数的性质

  1. 修改后的 Jensen 不等式
    对于准凸函数fff，有：

    0≤θ≤1⟹f(θx+(1−θ)y)≤max⁡{f(x),f(y)}.0 \leq \theta \leq 1 \implies f(\theta x + (1-\theta) y) \leq \max\{f(x), f(y)\}.0≤θ≤1⟹f(θx+(1−θ)y)≤max{f(x),f(y)}.

  2. 一阶条件（First-order condition）
    如果fff 是可微函数且定义域是凸集，则fff 是准凸的，当且仅当：

    f(y)≤f(x)⟹∇f(x)T(y−x)≤0.f(y) \leq f(x) \implies \nabla f(x)^T (y - x) \leq 0.f(y)≤f(x)⟹∇f(x)T(y−x)≤0.

    这一条件的几何解释：当yyy 在xxx 的等值面的一侧时，梯度方向与(y−x)(y-x)(y−x) 方向相反。
