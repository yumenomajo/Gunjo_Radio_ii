# 早上醒来

今天六点钟醒了，玩了一会红烧又睡着了。在困意中，做着迷迷糊糊的梦，隐约还记得一些梦里的颜色与事物：钢琴、教室、老同学… 人们似乎常常将梦与高深之道联系在一起，我也不例外。曾经的我也曾做过记录梦境来研究梦境的事情，不过也只是一个人自娱自乐罢。梦里真的有神灵一般的天道吗？我开始怀疑。

# 物理复习课

阴差阳错，我要开一次物理复习课了。实际上，我对这件事情并没有什么自信。其实我本来想讲讲数学分析的，除了例题之外还可以讲讲我之前研究的放缩技巧，还有相关难题。但是有同学觉得对着例题复习就足矣，我便接受一点意见选择了转换赛道。

为了准备好复习课，恐怕我要开始着手构建好一个物理知识体系。其实也可以重点讲讲老师让我们自习的部分，这也是一定程度上和课堂的互补。而那些经典的例题，其实也是课本上的那些基础，梳理一遍也就差不多了。

# 科研小进度

今天为了赶趟，一下午速通了视频数据信号转化为二值化矩阵，然后做了两个简单的分类算法；一个是超维特征向量＋范数法；另一个是简单的人工神经网路算法。

由于做得太快，实际上还有一些地方有所疏漏，为了组会应付好导师，到时候看看能不能类比引进上周末做好的 LSTM 模型，做时间序列的标签预测。周五的时候再抽出时间做做吧。

# 抽样调查

没错，我又在提我这一学期最大的敌人 ———— 抽样调查；今天大概非常粗糙地过完了所有知识点，深知自己一定是没有足够的时间和精力掌握好所有细节的证明了。那没办法，只能选择性放弃这些部分。现在的目标是：掌握所有最基本的计算，标准是课本上的习题全部会算；同时要知道估计量性质；不要求做复杂证明，掌握分层抽样和简单随机抽样部分证明即可。

所以，以后不要对着知识点内耗；直接上题目，边写边总结，做成一个大的知识网络就够了。类似的复习应该还要推广到最优化和概率论。时间不多了啊。

希望明天会更好吧。
