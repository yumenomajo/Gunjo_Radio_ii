# 3.3.2 比例分配

比例分配指的是按各层单元数占总单元数的比例，也就是按各层的层权进行分配。此时：

nhn=NhN=Wh或fh=nhNh=nN=f(3.19)\frac{n_h}{n} = \frac{N_h}{N} = W_h \text{ 或 } f_h = \frac{n_h}{N_h} = \frac{n}{N} = f \tag{3.19}nnh​​=NNh​​=Wh​ 或 fh​=Nh​nh​​=Nn​=f(3.19)

# 3.3.3 最优分配

分层随机抽样，将样本量分配到各层，

条件：在总费用给定，目标：估计量的方差达到最小

条件：估计量方差给定，目标：总费用最少

能满足这个条件的样本量分配就是最优分配。

如果我们考虑简单线性费用函数，总费用为：

C=c0+∑h=1Lchnh,(3.24)C = c_0 + \sum_{h=1}^{L} c_h n_h, \tag{3.24}C=c0​+h=1∑L​ch​nh​,(3.24)

则此时的最优分配是：

nhn=WhShch/∑h=1LWhShch=NhShch/∑h=1LNhShch,h=1,2,…,L.(3.25)\frac{n_h}{n} = \frac{W_h S_h}{\sqrt{c_h}} \Big/ \sum_{h=1}^{L} \frac{W_h S_h}{\sqrt{c_h}} = \frac{N_h S_h}{\sqrt{c_h}} \Big/ \sum_{h=1}^{L} \frac{N_h S_h}{\sqrt{c_h}}, \quad h = 1, 2, \dots, L. \tag{3.25}nnh​​=ch​​Wh​Sh​​/h=1∑L​ch​​Wh​Sh​​=ch​​Nh​Sh​​/h=1∑L​ch​​Nh​Sh​​,h=1,2,…,L.(3.25)

由公式 (3.25) 可以看出，如果某一层单元数较多、内部差异较大、费用比较省，则对这一层的样本量要多分配一些。

对于分层随机抽样，作为特例，如果每层抽样的费用相同，即 ch=cc_h = cch​=c 时，最优分配可简化为
