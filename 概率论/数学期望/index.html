# 4.1.1 数学期望的基本定义

# 定义

# 定义 4.1.1【简单随机变量】

对于简单随机变量 ξ\xiξ，称：

E(ξ)≜∑x∈ξ(Ω)xP(ξ=x)(4.1)\mathbb{E}(\xi) \triangleq \sum_{x \in \xi(\Omega)} x \mathbb{P}(\xi = x) \tag{4.1}E(ξ)≜x∈ξ(Ω)∑​xP(ξ=x)(4.1)

为 ξ\xiξ 的数学期望，简称为数学期望。

# 定义 4.1.2【非负随机变量】

对于任意随机变量 ξ≥0\xi \geq 0ξ≥0，称：

E(ξ)≜lim⁡n→∞(∑k=1n2nk−12nP(k−12n<ξ≤k2n)+nP(ξ>n))(4.3)\mathbb{E}(\xi) \triangleq \lim_{n \to \infty} \left( \sum_{k=1}^{n2^n} \frac{k-1}{2^n} \mathbb{P}\left(\frac{k-1}{2^n} < \xi \leq \frac{k}{2^n}\right) + n \mathbb{P}(\xi > n) \right) \tag{4.3}E(ξ)≜n→∞lim​(k=1∑n2n​2nk−1​P(2nk−1​<ξ≤2nk​)+nP(ξ>n))(4.3)

为随机变量 ξ\xiξ 的数学期望或均值。

# 定义 4.1.3【任意随机变量】

对于任意随机变量 ξ\xiξ，若 E(ξ+)<∞\mathbb{E}(\xi^+) < \inftyE(ξ+)<∞ 或 E(ξ−)<∞\mathbb{E}(\xi^-) < \inftyE(ξ−)<∞，则称：

E(ξ)≜E(ξ+)−E(ξ−)(4.4)\mathbb{E}(\xi) \triangleq \mathbb{E}(\xi^+) - \mathbb{E}(\xi^-) \tag{4.4}E(ξ)≜E(ξ+)−E(ξ−)(4.4)

为 ξ\xiξ 的数学期望或 ξ\xiξ 的均值；若 E(ξ+)=∞\mathbb{E}(\xi^+) = \inftyE(ξ+)=∞ 且 E(ξ−)=∞\mathbb{E}(\xi^-) = \inftyE(ξ−)=∞，则称 ξ\xiξ 的数学期望不存在。
