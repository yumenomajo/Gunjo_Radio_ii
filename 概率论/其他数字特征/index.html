# 定义

# 定义 4.2.1

将定义在 (Ω,F,P)(\Omega, \mathcal{F}, \mathbb{P})(Ω,F,P) 上的随机变量集合记为：

L2=L2(Ω,F,P)≜{ξ:E(ξ2)<∞},L^2 = L^2(\Omega, \mathcal{F}, \mathbb{P}) \triangleq \{\xi : \mathbb{E}(\xi^2) < \infty\},L2=L2(Ω,F,P)≜{ξ:E(ξ2)<∞},

为 L2L^2L2 空间。

# 定义 4.2.2

对于 ξ∈L2\xi \in L^2ξ∈L2，称：

D(ξ)≜E((ξ−E(ξ))2)(4.28)D(\xi) \triangleq \mathbb{E}((\xi - \mathbb{E}(\xi))^2) \tag{4.28}D(ξ)≜E((ξ−E(ξ))2)(4.28)

# 定义 4.2.3

若 ξ∈L2\xi \in L^2ξ∈L2，则称：

ξ∗=ξ−E(ξ)D(ξ)\xi^* = \frac{\xi - \mathbb{E}(\xi)}{\sqrt{D(\xi)}}ξ∗=D(ξ)​ξ−E(ξ)​

为 ξ\xiξ 的标准化。

标准化随机变量的均值为 0，方差为 1，是一个无量纲的量，它消除了度量单位的影响。

# §4.2.2 协方差与方差矩阵

# 定义

# 定义 4.2.4

设 ξ=(ξ1,ξ2,…,ξn)\xi = (\xi_1, \xi_2, \dots, \xi_n)ξ=(ξ1​,ξ2​,…,ξn​) 为随机向量，如果 ξk∈L2,1≤k≤n\xi_k \in L^2, \ 1 \leq k \leq nξk​∈L2, 1≤k≤n，称：

cov(ξi,ξj)≜E((ξi−E(ξi))(ξj−E(ξj)))(4.30)\text{cov}(\xi_i, \xi_j) \triangleq \mathbb{E}((\xi_i - \mathbb{E}(\xi_i)) (\xi_j - \mathbb{E}(\xi_j))) \tag{4.30}cov(ξi​,ξj​)≜E((ξi​−E(ξi​))(ξj​−E(ξj​)))(4.30)

为 ξi\xi_iξi​ 和 ξj\xi_jξj​ 的协方差，而称 n×nn \times nn×n 矩阵：

var(ξ)≜(cov(ξi,ξj))n×n(4.31)\text{var}(\xi) \triangleq (\text{cov}(\xi_i, \xi_j))_{n \times n} \tag{4.31}var(ξ)≜(cov(ξi​,ξj​))n×n​(4.31)

为 ξ\xiξ 的方差矩阵。

实际思考：

显然，cov(ξ,η)>0\text{cov}(\xi, \eta) > 0cov(ξ,η)>0 意味着：在概率加权的平均意义上：

(ξ−E(ξ))(η−E(η))>0,(\xi - \mathbb{E}(\xi)) (\eta - \mathbb{E}(\eta)) > 0,(ξ−E(ξ))(η−E(η))>0,

所以 cov(ξ,η)>0\text{cov}(\xi, \eta) > 0cov(ξ,η)>0 意味着 ξ\xiξ 和 η\etaη 有相同的变化趋势；

类似cov(ξ,η)<0\text{cov}(\xi, \eta) < 0cov(ξ,η)<0 意味着 ξ\xiξ 和 η\etaη 有相反的变化趋势。

协方差描述了两个随机变量之间相互关系的信息。

# 定理

# 定理 4.2.4

协方差有如下性质：

1° 对称性：cov(ξi,ξj)=cov(ξj,ξi)\text{cov}(\xi_i, \xi_j) = \text{cov}(\xi_j, \xi_i)cov(ξi​,ξj​)=cov(ξj​,ξi​)。

2° 对于任意实数 aaa，cov(aξi,ξj)=cov(ξi,aξj)=acov(ξi,ξj)\text{cov}(a\xi_i, \xi_j) = \text{cov}(\xi_i, a\xi_j) = a\text{cov}(\xi_i, \xi_j)cov(aξi​,ξj​)=cov(ξi​,aξj​)=acov(ξi​,ξj​)。

3° D(ξi)=cov(ξi,ξi)D(\xi_i) = \text{cov}(\xi_i, \xi_i)D(ξi​)=cov(ξi​,ξi​)。

4° cov(ξi,ξj)=E(ξiξj)−E(ξi)E(ξj)\text{cov}(\xi_i, \xi_j) = \mathbb{E}(\xi_i \xi_j) - \mathbb{E}(\xi_i)\mathbb{E}(\xi_j)cov(ξi​,ξj​)=E(ξi​ξj​)−E(ξi​)E(ξj​)。

5° D(ξi+ξj)=D(ξi)+D(ξj)+2cov(ξi,ξj)D(\xi_i + \xi_j) = D(\xi_i) + D(\xi_j) + 2\text{cov}(\xi_i, \xi_j)D(ξi​+ξj​)=D(ξi​)+D(ξj​)+2cov(ξi​,ξj​)。

6° 若 ξi\xi_iξi​ 与 ξj\xi_jξj​ 相互独立，则 cov(ξi,ξj)=0\text{cov}(\xi_i, \xi_j) = 0cov(ξi​,ξj​)=0，且：

D(ξi+ξj)=D(ξi)+D(ξj)。D(\xi_i + \xi_j) = D(\xi_i) + D(\xi_j)。D(ξi​+ξj​)=D(ξi​)+D(ξj​)。
