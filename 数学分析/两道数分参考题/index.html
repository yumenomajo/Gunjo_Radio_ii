思路梳理：

∫ξ2−ξQ(x)dx=ξ[Q(ξ)+Q(2−ξ)]2023.\int_{\xi}^{2-\xi} Q(x)\,dx = \frac{\xi[Q(\xi) + Q(2-\xi)]}{2023}.∫ξ2−ξ​Q(x)dx=2023ξ[Q(ξ)+Q(2−ξ)]​.

证明步骤：

  1. 构造辅助函数：
    构造一个函数

    F(ξ)=∫ξ2−ξQ(x)dx−ξ[Q(ξ)+Q(2−ξ)]2023.F(\xi) = \int_{\xi}^{2-\xi}Q(x)\,dx - \frac{\xi[Q(\xi)+Q(2-\xi)]}{2023}.F(ξ)=∫ξ2−ξ​Q(x)dx−2023ξ[Q(ξ)+Q(2−ξ)]​.

    我们希望证明存在 ξ\xiξ 使得 F(ξ)=0F(\xi)=0F(ξ)=0。

  2. 函数 F(ξ)F(\xi)F(ξ) 的连续性：
    QQQ 是连续函数，因此 ∫ξ2−ξQ(x)dx\int_{\xi}^{2-\xi}Q(x)\,dx∫ξ2−ξ​Q(x)dx 关于 ξ\xiξ 是连续的（积分上限下限均为连续变化，且积分核函数连续）。
    同时，Q(ξ)Q(\xi)Q(ξ) 与 Q(2−ξ)Q(2-\xi)Q(2−ξ) 也是连续函数，因此 ξ[Q(ξ)+Q(2−ξ)]\xi[Q(\xi)+Q(2-\xi)]ξ[Q(ξ)+Q(2−ξ)] 关于 ξ\xiξ 连续。
    故 F(ξ)F(\xi)F(ξ) 是 R\mathbb{R}R 上连续的函数。

  3. 在特定点上计算 F(ξ)F(\xi)F(ξ)：
    (1) 当 ξ=0\xi=0ξ=0 时：

    F(0)=∫02Q(x)dx−0[Q(0)+Q(2)]2023=∫02Q(x)dx.F(0) = \int_0^2 Q(x)\,dx - \frac{0[Q(0)+Q(2)]}{2023} = \int_0^2 Q(x)\,dx.F(0)=∫02​Q(x)dx−20230[Q(0)+Q(2)]​=∫02​Q(x)dx.

    因此有

    F(0)=∫02Q(x)dx,F(1)=−2Q(1)2023.F(0) = \int_0^2 Q(x)\,dx, \quad F(1) = -\frac{2Q(1)}{2023}.F(0)=∫02​Q(x)dx,F(1)=−20232Q(1)​.

  4. 利用中间值定理的思路：
    若 ∫02Q(x)dx\int_0^2 Q(x)\,dx∫02​Q(x)dx 和 −2Q(1)2023-\frac{2Q(1)}{2023}−20232Q(1)​ 异号，则 ∃ξ∈(0,1)\exists \xi \in (0,1)∃ξ∈(0,1) 使得 F(ξ)=0F(\xi)=0F(ξ)=0。

    • 如果 F(0)=0F(0)=0F(0)=0，则 ξ=0\xi=0ξ=0 本身即为解。
    • 如果 F(1)=0F(1)=0F(1)=0，则 ξ=1\xi=1ξ=1 本身为解。

    若上述两点都不满足，我们需要更进一步的讨论。

  5. 若 F(0)F(0)F(0) 与 F(1)F(1)F(1) 同号怎么办？
    情况分析：

      简言之：在区间 [0,2][0,2][0,2] 上积分为正，但 Q(1)Q(1)Q(1) 却为负。由于 QQQ 连续，这意味着 QQQ 在 [0,2][0,2][0,2] 内必有正区间与负区间的切换点，即存在 x0∈(0,2)x_0 \in (0,2)x0​∈(0,2) 使 Q(x0)=0Q(x_0)=0Q(x0​)=0。这为后面再次选择合适的 ξ\xiξ 提供可能。

    • 若 F(0)<0F(0) < 0F(0)<0 与 F(1)<0F(1) < 0F(1)<0 类似分析。

  1. 导出 QQQ 必存在正值的点：
    已知 ∫02Q(x)dx>0\int_0^2 Q(x)\,dx >0∫02​Q(x)dx>0。若 Q(x)Q(x)Q(x) 在整个区间 [0,2][0,2][0,2] 内均为非正值（即 Q(x)≤0Q(x)\leq 0Q(x)≤0 对任意 x∈[0,2]x\in[0,2]x∈[0,2] 成立），则有

    ∫02Q(x)dx≤∫020dx=0。\int_0^2 Q(x)\,dx \leq \int_0^2 0 \,dx = 0。∫02​Q(x)dx≤∫02​0dx=0。

    这与已知条件 ∫02Q(x)dx>0\int_0^2 Q(x)\,dx >0∫02​Q(x)dx>0 矛盾。因此，不能所有点都满足 Q(x)≤0Q(x)\leq 0Q(x)≤0。因此，存在至少一个点 x1∈[0,2]x_1 \in [0,2]x1​∈[0,2] 使得 Q(x1)>0Q(x_1)>0Q(x1​)>0。

  2. 比较点的位置与 x=1x=1x=1：
    由于我们知道 Q(1)<0Q(1)<0Q(1)<0，且 Q(x1)>0Q(x_1)>0Q(x1​)>0。有两种情况：

    • 情况 A：x1<1x_1 < 1x1​<1
    • 情况 B：x1>1x_1 > 1x1​>1

    （若 x1=1x_1=1x1​=1 则 Q(1)>0Q(1)>0Q(1)>0 与已知 Q(1)<0Q(1)<0Q(1)<0 矛盾，故不可能。）

  3. 利用连续性与介值定理：
    无论是情况 A 还是情况 B，本质上是说 QQQ 在区间 [0,2][0,2][0,2] 上有一个点取正值，有一个点取负值（即 Q(1)<0Q(1)<0Q(1)<0）。由于 QQQ 是连续函数，连续函数在区间内的值变化必须遵从介值定理：

    • 情况 A（x1<1x_1<1x1​<1 且 Q(x1)>0,Q(1)<0Q(x_1)>0, Q(1)<0Q(x1​)>0,Q(1)<0）：
      在 [x1,1][x_1,1][x1​,1] 上，Q(x1)>0Q(x_1)>0Q(x1​)>0 且 Q(1)<0Q(1)<0Q(1)<0。
      由于 QQQ 连续，则当从 x1x_1x1​ 移动到 1 时，Q(x)Q(x)Q(x) 从正值变化到负值。
      连续函数由正变负的过程中必定经过零点，即存在 x0∈(x1,1)⊂(0,2)x_0 \in (x_1,1)\subset(0,2)x0​∈(x1​,1)⊂(0,2) 使 Q(x0)=0Q(x_0)=0Q(x0​)=0。

    • 情况 B（x1>1x_1>1x1​>1 且 Q(x1)>0,Q(1)<0Q(x_1)>0, Q(1)<0Q(x1​)>0,Q(1)<0）：
      在 [1,x1][1,x_1][1,x1​] 上，Q(1)<0Q(1)<0Q(1)<0 且 Q(x1)>0Q(x_1)>0Q(x1​)>0。
      同理，由 QQQ 的连续性可知，Q(x)Q(x)Q(x) 从负变正也必须经过零点，即存在 x0∈(1,x1)⊂(0,2)x_0 \in (1,x_1)\subset(0,2)x0​∈(1,x1​)⊂(0,2) 使 Q(x0)=0Q(x_0)=0Q(x0​)=0。

  4. 总结：
    无论正值点位于 1 的左侧还是右侧，都会得到在区间 (0,2)(0,2)(0,2) 内有一个零点 x0x_0x0​，从而在 [0,2][0,2][0,2] 上出现 “正 - 负” 或 “负 - 正” 的切换点。
